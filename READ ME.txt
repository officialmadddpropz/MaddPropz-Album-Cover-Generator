Here are install instructions for the album cover generator script.
Install Instructions — Album Cover Generator
1. Install Python (if you don't have it)

Windows: Download from python.org/downloads — check ✅ "Add Python to PATH" during install.
Mac: Python usually comes pre-installed. Check with python3 --version in Terminal. If missing, install from python.org or run brew install python3 (if you have Homebrew).
Linux: Usually pre-installed. If not: sudo apt install python3 python3-pip

2. Install the required library (Pillow)
Open Terminal (Mac/Linux) or Command Prompt/PowerShell (Windows) and run:

pip install Pillow    < in command window on your windows computer

If that fails on Mac/Linux with a permissions error, try:

pip3 install --user Pillow

3. Set up your project folder
Create a folder (e.g. AlbumCovers) and put two things in it:

album_cover_generator.py (the script I gave you)

IMPORTANT!!!!!!

Your photo — name it me.jpg (or any image; the script will auto-find it if me.jpg isn't there)

4. Run it
In Terminal/Command Prompt, navigate to that folder:

cd path/to/AlbumCovers
python3 album_cover_generator.py
(On Windows, you may need python instead of python3.)

5. Check your results
Four files will appear in the same folder:

style_oldschool_hiphop.jpg
style_2026_premium_cinema.jpg
style_vintage_vinyl.jpg
style_dark_cyberpunk.jpg

Optional — better fonts
The script looks for real bold fonts on your system and fakes extra weight if it can't find a true "Black/Impact" style font. If you own a .ttf/.otf Impact-style font, drop it in the same folder and tell me its filename — I'll point the script directly at it for an even cleaner look.
